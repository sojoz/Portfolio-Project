Patisserie Project

This is the first project that I ever did through coding. In this you can see how I am still experimenting with margins and padding. I learned how to use CSS, HTML and some javascript. For this project, I was meant to create a landing page for something that I enjoy and I had to go for pastries! I also had fun writing the content myself. I was able to create a basic web page for a 4 week course. At the bottom of the page, there are links to various pages with information on different pastries. There is also a button which allows the user to take a quiz, through alerts, to find out which pastry they are based on some questions!
